# 42.scop
The main goal of this project is to learn a bit about OpenGL. This will be able to print a 3D object (with the .obj extension) and rotate it around the main axis.

# INPUTS
## Render inputs
	P:
		Switch between 3 rendering modes (Filled, Wireframe, Point)

	T:
		Toggle texturing

## Movement inputs
	R:
		Toggle auto rotation

	PAGE UP / PAGE DOWN:
		Move the object on the Y axis

	UP / DOWN:
		Move the object on the Z axis

	LEFT / RIGHT:
		Move the object on the X axis
